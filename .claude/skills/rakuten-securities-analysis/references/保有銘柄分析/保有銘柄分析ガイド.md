# 保有銘柄分析ガイド

## 概要

楽天証券で保有している銘柄について、ファンダメンタル分析とテクニカル分析の観点から評価するためのガイド。
国内株式・米国株式・中国株式（香港・上海・深セン）・投資信託を対象とする。

**できること:**

- ファンダメンタル分析（PER・PBR・ROE・売上成長率・利益率など）
- テクニカル分析（トレンド・支持線/抵抗線・各種指標）
- 同業他社比較と業界内ポジショニング
- リスク評価と投資シナリオの整理
- 構造化された分析レポートの作成

**できないこと（制約）:**

- リアルタイム株価・市場データへのアクセス（外部のデータソースを参照すること）
- 投資の助言・推奨
- 将来の運用成果の保証
- 非公開・未発表の取引データやインサイダー情報の利用
- 公開情報以外に基づく分析（バリュエーションモデルは前提と限界を伴う）

---

## 使い方

### ステップ1: 銘柄情報の提供

次のいずれかを伝える:

- 証券コード／ティッカー（例: 7974（任天堂）、AAPL、0700.HK、600519.SH）
- 市場区分（国内株式／米国株式／中国株式）
- または財務データを直接渡す

### ステップ2: 分析タイプの選択

依頼で範囲が指定されていなければ、**詳細分析（総合的なファンダメンタル分析）を既定**とする。ユーザーが軸を明示した場合のみ下記に切り替える:

- **クイック分析**: 主要指標の概観だけでよいとき
- **テクニカルレビュー**: チャートパターンと各種指標を中心に求められたとき
- **同業他社比較**: 競合との比較を主目的とするとき

### ステップ3: 分析レポートの受領

次の構成で構造化したレポートを返す:

- 企業概要
- 主要な財務指標
- バリュエーション評価
- リスク要因
- 投資シナリオ

## 分析フレームワーク

### ファンダメンタル分析の指標

#### バリュエーション指標

| 指標      | 計算式                    | 解釈                                       |
| --------- | ------------------------- | ------------------------------------------ |
| PER       | 株価 ÷ EPS                | 15未満は割安、25超は割高                   |
| PBR       | 株価 ÷ 1株あたり純資産    | 1未満は割安余地、3超は成長織り込み済み     |
| PSR       | 株価 ÷ 売上高             | 赤字企業の評価に有用                       |
| EV/EBITDA | 企業価値 ÷ EBITDA         | 10未満は魅力的、15超は割高                 |
| PEG       | PER ÷ 利益成長率          | 1未満は成長に対して割安                    |

#### 収益性指標

| 指標       | 計算式                     | 目安                         |
| ---------- | -------------------------- | ---------------------------- |
| 売上総利益率 | 売上総利益 ÷ 売上高       | 40%超で高い価格決定力        |
| 営業利益率 | 営業利益 ÷ 売上高          | 15%超で効率的な事業運営      |
| 純利益率   | 当期純利益 ÷ 売上高        | 10%超で健全な収益性          |
| ROE        | 当期純利益 ÷ 自己資本      | 15%超で良好な資本効率        |
| ROIC       | 税引後営業利益 ÷ 投下資本  | WACC超で価値創造             |

#### 成長性指標

| 指標         | 着目点                                       |
| ------------ | -------------------------------------------- |
| 売上成長率   | 前年比10%超で強い需要                        |
| EPS成長率    | 売上成長を上回るのが望ましい（営業レバレッジ） |
| FCF成長率    | 持続的な成長ドライバー                       |

#### 財務健全性

| 指標         | 計算式                       | 安全圏               |
| ------------ | ---------------------------- | -------------------- |
| 負債資本倍率 | 総負債 ÷ 自己資本            | 多くの業種で1未満    |
| 流動比率     | 流動資産 ÷ 流動負債          | 1.5超で健全な流動性  |
| インタレスト・カバレッジ・レシオ | EBIT ÷ 支払利息 | 5超で余裕あり        |
| FCF利回り    | FCF ÷ 時価総額               | 5%超で魅力的         |

### テクニカル分析のパターン

#### トレンド指標

- **移動平均線**: 50日線・200日線のクロス
- **MACD**: モメンタムとトレンド方向
- **RSI**: 買われすぎ（70超）／売られすぎ（30未満）

#### チャートパターン

- **支持線／抵抗線**: 重要な価格水準
- **ヘッドアンドショルダー**: 反転パターン
- **ダブルトップ／ダブルボトム**: 反転シグナル
- **ブレイクアウト**: 出来高を伴う値動き

## ポートフォリオ全体を評価するときの観点（楽天証券CSV固有）

個別銘柄ではなく「ポートフォリオ全体を見て」「リスクを評価して」と頼まれた場合、汎用的な分散の話だけで終わらせない。楽天証券の資産残高CSVには、素の知識だけでは見落としがちな固有情報が乗っている。これらを必ず拾って、その口座のデータに即した指摘をすること。一般論に流れると、CSVを渡した意味が薄れる。

- **実質の国・地域／通貨エクスポージャー**: 「投資信託」種別の金額を額面どおり受け取らない。S&P500・NASDAQ-100・FANG+ は中身がほぼ米国株、オルカン（オール・カントリー）も時価の6割前後が米国。これらと米国個別株を合算した「実質米国比率」を概算し、地域分散が効いているかを評価する。外貨建MMF・外貨預り金・米国株は為替（USD/JPY 等、CSV末尾の参考為替レートに当日値がある）変動の影響を受ける点も併せて見る。
- **ファンド間の中身の重複**: 別名のファンドでも構成銘柄が重なることが多い（例: S&P500 と NASDAQ-100 と FANG+ は大型テックで重複、オルカンにも同じ銘柄が含まれる）。「銘柄数が多い＝分散」ではないため、テーマ・セクターの二重露出を指摘する。
- **NISA枠の使い分け**: CSVの「口座」列に NISA成長投資枠／つみたて投資枠／旧つみたてNISA などが入っている。同一ファンドが複数の枠に分かれて保有されていることもある。非課税枠の活用状況や、枠ごとの商品配置の妥当性に触れられるとよい。
- **集中と含み損益の偏り**: 評価損益[％]列から、特定銘柄への利益集中（高含み益＝相対的に下落耐性が低い）や、孤立した含み損銘柄を読み取る。1銘柄が資産全体に占める割合（特に投信1本で過半など）は集中リスクとして明示する。
- **現金比率と待機資金**: 預り金・楽天銀行普通預金残高から、相場下落時に動かせる現金の厚みを評価する。

これらはCSVの数値から概算で導く補助計算であり、CSVに元々無い値（実質米国比率など）を出すときは「概算」「指数構成の一般的な前提に基づく」と明示する。元CSVの金額・損益率そのものは再計算・丸めをしない方針（変換タスクと共通）は守ること。

## より良い結果を得るためのヒント

1. 直近の財務データがあれば提供すると、より正確な分析ができる
2. 投資期間（短期トレード／長期投資）を明示する
3. 特に気になっている懸念点を伝える
4. 比較したい同業他社を挙げる
5. 相場環境（強気相場／弱気相場）を伝える

（制約事項は冒頭「できないこと（制約）」を参照。本分析は専門家による金融アドバイスの代替にはならない。）
