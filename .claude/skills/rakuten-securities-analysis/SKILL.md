# 楽天証券 資産状況分析

楽天証券の資産状況を扱うスキル。
依頼の内容に応じて、下記対応する参照ファイルを読み込んでから作業する。

## CSV → Markdown 変換

- 概要：楽天証券からダウンロードした資産残高CSV（CP932 / Shift-JIS）を、`楽天証券_資産状況_yyyymmdd.md` というレポートに整形する。
- 参照先：`references/CSV2マークダウン/CSV変換手順.md`

## 保有銘柄分析

- 概要：保有している個別銘柄について、ファンダメンタル分析（PER・PBR・ROE・売上成長率・利益率など）やテクニカル分析を行い、構造化された分析レポートを作成する。
- 参照先：`references/保有銘柄分析/保有銘柄分析ガイド.md`を参照する。
